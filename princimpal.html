BASE DE DADOS

Base de dados: São um conjunto de dados que visam organizar uma informação.

Dados: Valor de um campo armazenado matéria prima para obtênção de informação.



Informações: Qualquer estruturação organizada destes dados, compilados e processados de acordo com a solicitação de consultas


MODELO DE DADOS


Modelo de dados: São fórmas de representção de dados existentes em uma base de dados (BD)

Elas descrevem a extrutura das informações contidas na BD


    TIPOS DE MODELOS DE DADOS


  1. M. Hierárquico
  2. M. Relacional
  3. M. Orientado a obectos
